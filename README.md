# AetherLauncher

A fast, modern Minecraft launcher built for **performance** and **zero-friction modding**.

> Install mods instantly. No setup. No hassle.

> Built Upon <a href="https://github.com/antunnitraj/OfflineMinecraftLauncher">OfflineMinecraftLauncher</a> 

---

## 🚀 Features

* ⚡ **One-click play** — launch instantly
* 🧩 **Instance-based system** — fully isolated profiles
* 📦 **Mod installer (Modrinth)** — search & install directly
* 🔗 **Automatic dependency resolution** — no manual fixing
* 🎨 **Clean, modern UI** — minimal and fast
* 🧠 **Performance-focused design** — optimized out of the box

---

## 💡 Why Aether Launcher?

* Faster setup than traditional launchers
* No confusing menus or bloated UI
* Mods just work — dependencies handled automatically
* Built for speed, not complexity

## 📦 Download

👉 Go to the [Releases](../../releases) page and download the latest version.

### Windows

* Download the `.exe` or build the MSI from `packaging/build-windows-installer.cmd`
* Double click the `.msi` to install

### Linux

* Download the binary or install the `.deb`

```bash
chmod +x death-client-linux
./death-client-linux
```

### Windows MSI

If you want a proper Windows installer, run `packaging/build-windows-installer.cmd` on Windows after installing the WiX CLI, or run `packaging/build-windows-installer.ps1` after publishing `win-x64`.

If you do not have Windows, use GitHub Actions: run the workflow in `.github/workflows/windows-msi.yml` from the Actions tab, then download the MSI artifact from that workflow run.
